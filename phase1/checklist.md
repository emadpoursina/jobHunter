Before selecting countries, these hard blockers must be acknowledged:

These countries are **excluded** from the priority list regardless of tech opportunity.

## Step 1: Priority country selection

### Scoring criteria

Each country is scored on 5 factors (1–5 scale):

| Factor | Description |
|---|---|
| Visa Accessibility | How easy it is for an Iranian national to obtain a work visa |
| Tech Job Market | Demand for Full-Stack JS / Node.js / React developers |
| Salary vs. Cost of Living | Real purchasing power after living expenses |
| English-Friendly Work | Can you work professionally in English? |
| Path to PR | How clear and fast is the permanent residency pathway? |
