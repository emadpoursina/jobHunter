# Generated CVs

Tailored CVs and tailoring reports from the [CV generator agent](../../docs/agents/cv-generator.md).

**Naming:** `CV_<CompanySlug>_<JobTitleSlug>_<YYYY-MM-DD>.md`

**Stages:** `draft` (AI output) → `reviewed` (human approved) → linked in `applications/pipeline.md` when sent.

Do not send a CV until the human review checklist in the tailoring report is complete.
